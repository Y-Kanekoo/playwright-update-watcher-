# CLAUDE.md

このリポジトリではグローバル設定を継承し、以下のプロジェクト固有Intentのみを追加します。

## Intent: Playwrightリリース監視の信頼性

GitHub Releases APIの取得、Discord通知、状態ファイル更新の順序を明確に保ちます。通知に失敗した場合は状態ファイルを更新せず、次回実行で再検知できる状態を維持します。

## Intent: 言語統一（日本語）

ユーザー向けの説明、ログ、エラー、ワークフロー名、ステップ名は日本語で統一します。外部サービス仕様で固定されるキー名や環境変数名は例外として扱います。

## Intent: 依存最小化

Python実装は標準ライブラリのみを使い、外部パッケージや追加のプロジェクト設定ファイルを導入しません。
